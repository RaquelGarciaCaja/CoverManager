# Cover Manager 
Herramientas usadas: React, HTML, CSS3 (SASS), react router dom, react-zoom-pan-pinch

## frameworks usado: REACT
### Instalación 🔧

para ponerlo en marcha en su dispositivo:
```bash
npm install
```

para arrancar el proyecto:
```bash
npm start 
```

para publicarlo en GitHub Pages:
```bash
npm run build
```
